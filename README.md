# Fitness App V1

A free phone-first fitness tracker for calories, macros, weight, and lifts.

## Run Locally

```sh
npm start
```

Then open:

```text
http://localhost:4173
```

## Free iPhone Use

Deploy this folder to GitHub Pages, then open the HTTPS GitHub Pages URL on iPhone Safari and choose Share -> Add to Home Screen.

The app stores data on the device with browser storage, so no paid database or API key is required.
